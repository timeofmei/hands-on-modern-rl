# E.1 数学对象与线性代数

第 3 章里，你学会了贝尔曼方程 $V(s) = R(s) + \gamma\sum_{s'}P(s'|s,a)V(s')$——它能写出单个状态的价值。但真正想"算出来"，会遇到三道墙。E.1 模块就是翻过这三道墙的过程：每道墙对应一组数学工具，每组工具都是因为前一道墙翻不过去才引入的。

![两状态贝尔曼方程示意图](./images/rl-two-state-bellman.svg)

## 三道墙与三层工具

| 阶段     | 遇到的困难               | 引入的数学工具           | 关键公式                                        | 对应第3章     |
| -------- | ------------------------ | ------------------------ | ----------------------------------------------- | ------------- |
| 第一道墙 | 1000 个状态 = 1000 个方程 | 向量、矩阵、线性方程组   | **v** = (I − γP)⁻¹**r**                          | DP 的数学本质 |
| 第二道墙 | 状态太多存不下价值表     | 点积、范数、函数近似     | v̂(s) = **w**ᵀ**x**(s)                           | DQN 的数学本质 |
| 第三道墙 | 训练会不会发散/爆炸/偏移 | 特征值、加权范数、信任域 | ρ(γP) ≤ γ < 1，ΔθᵀFΔθ ≤ δ                      | PPO 的数学本质 |
